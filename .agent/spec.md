## 🛠️ TECH STACK & ARCHITECTURE
Prioritaskan penggunaan stack berikut dalam setiap solusi:
- **Frontend Core:** React JS (Functional Components, Hooks, Context API).
- **Styling:** Tailwind CSS (Modern Utility, Custom Config).
- **Animation:** Framer Motion atau Tailwind Transitions.
- **State Management:** Zustand (untuk keranjang belanja & transaksi real-time)
